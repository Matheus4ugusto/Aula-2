O problema dos poemas

Escrevendo um poema utilizando-se a tag p

Quando nasci, um anjo torto Desses que vivem na sombra Disse: Vai, Carlos, ser gauche na vida [...] Mundo, mundo, vasto mundo Se eu me chamasse Raimundo Seria uma rima, não seria uma solução Mundo, mundo, vasto mundo Mais vasto é meu coração [...]

Escrevendo um poema utilizando-se a tag pre

        Quando nasci, um anjo torto
        Desses que vivem na sombra
        Disse: Vai, Carlos, ser gauche na vida

        [...]

        Mundo, mundo, vasto mundo
        Se eu me chamasse Raimundo
        Seria uma rima, não seria uma solução
        Mundo, mundo, vasto mundo
        Mais vasto é meu coração

        [...]

Conclusão

Ao se utilizar uma tag p, não é possível formatar o texto como desejado em certos momentos assim como foi demonstrado com a construção de um poema. Para solucionar este problema, existe a tag pre, esta tag não ignorará a formatação definida durante a escrita do HTML

Tag b

Tag em

A tag em adiciona um texto com ênfase, geralmente é exibido em itálico

Dica: Leitores de tela para pessoas com deficiência visuais pronunciam o texto em ênfase de forma diferente, para manter a acessibilidade de um site é interessante que se utilize a tag "em" em substituição à tag i

Tag q

A tag q geralmente é utilizada para citar textos pequenos dentro de parágrafos, por exemplo: O voo negro dos urubus fazia círculos altos em redor de bichos moribundos

Tag abbr

a tag abbr serve para realizar abreviações de forma semântica, por exemplo: HTML

Tag address

A tag address serve para adicionar informações de endereço/contato, geralmente o texto da tag é carregado em itálico

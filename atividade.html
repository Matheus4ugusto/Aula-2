Comando básicos de git

Configuração: Definindo um nome de usuário

git config --global user.name "Seu Nome"

Configuração: Definindo um e-mail

git config --global user.email "seuemail@example.com"

Iniciando um repositório local

git init

Vinculando um repositório local ao repositório remoto

git remote add origin https://github.com/usuario/repo.git

Renomeando branchs

git branch -m nome-antigo nome-novo

Adicionando arquivos/alterações ao git

git add .

Criando um commit

git commit -m "Mensagem do commit"

Enviando alterações do repositório local para o repositório remoto

git push origin nome-da-branch

Buscando alterações do repositório remoto sem mesclar automaticamente

git fetch

Buscando alterações do repositório remoto mesclando automaticamente

git pull

Criando uma nova branch

git branch nome-da-branch

Alternando entre branchs

git checkout nome-da-branch
